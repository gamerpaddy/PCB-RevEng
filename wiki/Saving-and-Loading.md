# Saving and Loading

## Automatic saving

Your work is saved inside the browser automatically every few seconds, including the photos. The text on the far right of the top bar shows the status, such as saving or saved a moment ago. When you reload the page your last session, including its images, comes back on its own.

This automatic copy lives in this browser on this computer. It is not a file you can move. Use Save to make a portable backup.

## Saving a project file

Click **Save** in the top bar, or press Ctrl and S. A project file with the .pcbrev.json ending is downloaded. This file contains everything, including the photos, so it is a complete backup you can store or share.

## Opening a project file

Click **Open** in the top bar, or press Ctrl and O, and choose a project file. You can also drag a project file onto the canvas. The board, the photos and the nets are restored, and the opened project becomes the new automatically saved session.

## Starting a new project

Click **New** in the top bar. You are asked to confirm because this clears the current board. The automatically saved session is cleared as well, so the next reload starts empty and the welcome box appears again.

## The sample project

The sample project is a finished board you can study.

- On the first run, click **Load sample project** in the welcome box.
- At any later time, click the **?** button in the top bar and then click **Load sample project**.

Loading the sample replaces the current board, so save first if you want to keep your work. Once the sample is loaded it is saved automatically like any other project, including its photos, so it survives a reload.
